# Flipside 빌드 체크

데모 전 빠른 상태 점검을 수행합니다.

## 점검 항목

다음을 순서대로 실행하세요:

1. **TypeScript 타입 체크**: `npx tsc --noEmit` 실행
2. **빌드 테스트**: `npm run build` 실행
3. **환경 변수 확인**: `.env.local`에 `GEMINI_API_KEY`가 설정되어 있는지 확인

## 결과 보고

점검 결과를 다음 형식으로 보고하세요:

```
체크 결과
━━━━━━━━━━━━━━━━━━
타입 체크  : OK / FAIL (에러 N개)
빌드      : OK / FAIL
환경 변수  : OK / MISSING
━━━━━━━━━━━━━━━━━━
```

실패한 항목이 있으면 원인과 수정 방법을 간단히 안내하세요.
빌드 에러가 있으면 수정할지 사용자에게 물어보세요.
